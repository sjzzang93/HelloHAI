# HAI 검색 서비스

## 🚀 577.kr 스타일 빠른 접속 서비스

### 주요 기능
- **8개 카테고리** 바로가기 (맛집, 쇼핑, 뉴스, 블로그 등)
- **실시간 크롤링** (네이버, 구글, 다음, 빙 등)
- **모바일 최적화** 반응형 디자인
- **24시간 운영** 클라우드 서버

### 파일 구조
- `index.html` - 메인 검색 페이지
- `landing.html` - 577.kr 스타일 랜딩 페이지
- `server.js` - Node.js 서버 (크롤링 엔진)
- `package.json` - 의존성 관리

### 배포
Railway를 통한 클라우드 배포 지원
