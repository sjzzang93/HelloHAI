## 🏆 **로컬에서 실행 + ngrok** (1분!)

### ✨ 장점
- ✅ **1분이면 완료**
- ✅ **설치 필요 없음**
- ✅ **바로 사용 가능**

### 방법
1. **서버 실행**
   ```bash
   npm start
   ```

2. **ngrok 설치 및 실행**
   ```bash
   brew install ngrok
   ngrok http 3000
   ```

3. **완료!**
   - ngrok이 URL 제공
   - 전 세계 어디서나 접속 가능
   - 컴퓨터 켜 있어야 함
